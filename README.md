## Development

### Getting Setup

**Prerequisites: Ensure you have a version of `node.js` > 4.0.0**

1. Clone the repository
2. Run `npm install`

### Building/Running

#### Building
```
npm run build
```

#### Running
```
npm start
```

#### Build and start
```
npm run freshstart
```
